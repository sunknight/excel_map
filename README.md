# Excel测试用例脑图生成器

一个用于将Excel测试用例文件转换为可视化脑图的Web应用。支持上传包含测试用例的Excel文件，自动解析数据并生成交互式脑图，方便测试人员查看和管理测试用例。

## 功能特性

### 核心功能
- 📁 **Excel文件上传**: 支持.xlsx和.xls格式的Excel文件
- 🔍 **智能解析**: 自动解析Excel中的测试用例数据
- 🌳 **脑图生成**: 按照功能-子功能结构生成可视化脑图
- 🖱️ **交互操作**: 支持节点拖拽、画布缩放、平移等交互
- 📋 **详情查看**: 点击叶子节点查看完整的测试用例信息
- ⚙️ **字段配置**: 支持自定义字段映射，灵活适配不同Excel格式

### 交互功能
- 🔍 **缩放控制**: 
  - 支持放大、缩小、重置缩放
  - 触控板双指缩放（平滑缩放体验）
  - 缩放范围：30% - 200%
- 🎯 **自动布局**: 智能计算节点位置，自动优化布局
- 📐 **适应屏幕**: 一键将脑图适应屏幕显示
- 🔄 **展开/折叠**: 
  - 支持展开所有或折叠所有节点
  - 点击父节点展开/折叠子节点
  - 折叠展开时节点位置保持固定
- 📋 **复制画布**: 将脑图复制到剪贴板（支持粘贴到其他应用）
- 💾 **下载PNG**: 将脑图导出为高清PNG图片

### 安全特性
- 🛡️ **XSS防护**: 所有用户输入经过转义处理，防止注入攻击
- 🔒 **内容安全**: 使用安全的DOM API，无innerHTML注入风险

## Excel文件格式要求

### 必需字段
- `编号`: 测试用例编号
- `功能`: 功能模块名称
- `子功能`: 子功能名称
- `用例名称`: 测试用例名称

### 可选字段
- `模块`: 模块名称
- `优先级`: 优先级（P1/P2/P3）
- `业务流程`: 业务流程描述
- `前置条件`: 前置条件
- `步骤描述`: 测试步骤
- `预期结果`: 预期结果
- `端类型`: 端类型（如：移动+PC）
- `备注`: 备注信息
- `维护人`: 维护人信息

## 安装和运行

### 方法1: 使用本地服务器（推荐）

1. 安装Node.js和npm
2. 进入项目目录：
```bash
cd excel_map
```

3. 安装依赖（可选，仅用于开发）：
```bash
npm install
```

4. 启动本地服务器：
```bash
npm start
```

5. 在浏览器中打开: http://localhost:38195

### 方法2: 直接打开文件

直接在浏览器中打开 `index.html` 文件即可使用。

注意：由于浏览器的CORS限制，某些功能可能需要通过HTTP服务器运行。

## 使用说明

### 基本使用流程

1. **上传Excel文件**
   - 点击"上传Excel文件"按钮
   - 选择包含测试用例的Excel文件
   - 系统自动解析并生成脑图

2. **选择工作表**（如果Excel有多个Sheet）
   - 从下拉菜单中选择要可视化的Sheet
   - 系统自动尝试匹配字段
   - 如果字段不匹配，会清空画布并提示配置

3. **配置字段**（如需要）
   - 如果自动匹配失败，点击"配置字段"按钮
   - 为Level 1-4分别选择对应的Excel列
   - Level 2和Level 3为可选
   - 点击"生成脑图"

4. **查看和操作脑图**
   - 脑图按照"功能 → 子功能 → 测试用例"的结构展示
   - 绿色渐变色区分不同层级
   - 节点后显示用例数量统计（括号内）

5. **查看测试用例详情**
   - 点击最底层的测试用例节点
   - 弹窗显示该测试用例的完整信息
   - 点击关闭按钮或按ESC键关闭弹窗

### 交互操作

- **拖拽节点**: 按住节点可拖拽调整位置
- **缩放画布**: 
  - 使用右上角 +/- 按钮
  - 触控板双指缩放手势（按住Ctrl/Cmd + 滚轮）
- **平移画布**: 
  - 按住画布空白处拖拽
  - 使用触控板双指滑动或鼠标滚轮
- **展开/折叠**: 
  - 点击父节点前的图标
  - 使用顶部控制按钮
- **查看详情**: 点击叶子节点（最底层的测试用例）

### 快捷键

- `ESC`: 关闭详情弹窗

## 脑图配色方案

脑图采用绿色系渐变配色，清新自然：

- **一级节点（根节点）**: #3E7C17（深绿）
- **二级节点（功能）**: #66B240（中绿）
- **三级节点（类型）**: #A3D977（浅绿）
- **四级节点（子功能）**: #F0F9E6（极浅绿）
- **五级节点（测试用例）**: 白色背景，深绿文字

## 技术栈

- **前端框架**: 原生HTML/CSS/JavaScript
- **Excel解析**: XLSX.js (SheetJS)
- **图形渲染**: SVG + DOM
- **布局算法**: 树形结构自动布局
- **图片导出**: html2canvas

## 注意事项

1. **文件格式**: 仅支持.xlsx和.xls格式的Excel文件
2. **数据大小**: 建议单个Excel文件不超过10MB
3. **字段要求**: 必须包含编号、功能、子功能、用例名称四个字段
4. **编码格式**: Excel文件建议使用UTF-8编码
5. **安全**: 应用已实施XSS防护，可安全处理用户上传的Excel数据

## 常见问题

### Q: 上传Excel文件后没有反应？
A: 请检查：
- Excel文件格式是否正确（.xlsx或.xls）
- 是否包含必需的字段（编号、功能、子功能、用例名称）
- 浏览器控制台是否有错误信息

### Q: 脑图显示不完整？
A: 尝试：
- 点击"适应屏幕"按钮
- 使用缩放功能调整显示比例
- 拖拽画布查看其他部分

### Q: 节点位置可以手动调整吗？
A: 可以，按住节点拖拽即可调整位置。

### Q: 如何导出脑图？
A: 
- 点击"复制画布"按钮复制到剪贴板
- 点击"下载PNG"按钮保存为图片文件
- 导出时不会看到视觉跳动，体验流畅

### Q: 缩放功能如何使用？
A: 
- 使用右上角的 +/- 按钮
- 使用触控板双指缩放手势（Mac: 捏合手势，Windows: Ctrl+滚轮）
- 缩放以鼠标位置为中心，体验流畅

### Q: 折叠展开节点时会跳动吗？
A: 不会，已优化折叠展开体验，节点位置保持固定。
